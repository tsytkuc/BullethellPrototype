This prototype runs through a C# backend

The current file:// version is only a guide page. To view the actual bullet hell sample, start the ASP.NET Core app and open http://localhost:5000.

Run steps:

  1. Install the .NET 8 SDK
  2. Run dotnet restore in this folder
  3. dotnet run
  4. Open http://localhost:5000

See README.md for more details.
